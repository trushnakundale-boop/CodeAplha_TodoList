📌 Project Overview

TaskFlow is a modern To-Do List Web Application developed using HTML, CSS, and JavaScript.
It helps users organize daily tasks, track progress, and manage productivity efficiently through an interactive interface.

This project demonstrates frontend development skills such as DOM manipulation, responsive UI design, and local storage data management.

---

🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Task priority (High, Medium, Low)
- Task filtering (All, Completed, Pending)
- Task counter (Total, Completed, Remaining)
- Search tasks functionality
- Save tasks using LocalStorage
- Responsive design for mobile and desktop
- Dark theme modern interface

---

🛠 Technologies Used

Technology| Usage
HTML5| Page structure
CSS3| Styling and responsive layout
